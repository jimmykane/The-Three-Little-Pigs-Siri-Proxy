Get Connected Now

This server is 100% compatible with the Spire GUI. A guide on how to get the Spire GUI can be found here.
After you got the GUI running up and well then you will need a Spire Proxy address.
Spire Proxy address: https://jimmykane.dyndns-at-home.com (note that it uses https and there is no port number)

Youu will also need a certificate in order to make a secure connection to the server
You can downloade the certificate here: global.crt.
Open it with Mobile Safari to install. If you are having trouble then download it via PC/Max/Linux and email it to yourself.
